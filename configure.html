Configuring the System App

All server settings should be specified on the file on the root of the application. The specific settings are all described on this file.

The following settings will certainly need your attention:

Settings.general
  • Sets the desired title for the app, something like "My Company System".
  • This should be set to false on production environments.
Settings.database
  • MongoDB connection string. Default is host "localhost", database "systemapp".
Settings.web
  • Set to true if you're deploying to common PaaS services. More info on the next section.
  • The port used by the Node.js server, default is 3003.
