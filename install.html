Installing the System App

The NPM way

No frills installation. If you have Node.js installed you can simply get the System App:

$ sudo npm install systemapp -g

The automated way

Simply download the script, make it executable and run!

$ wget https://raw.github.com/zalando/system/master/install.sh
$ chmod +x install.sh
$ ./install.sh

Manual installation

If you want to install it manually, make sure you have these on your system:

  • Node.js (0.8.x +)
  • MongoDB (1.8.2 +)
  • ImageMagick (optional to generate map thumbnails)

1. Create a new folder "systemapp":

$ mkdir systemapp
$ cd systemapp

2. Download the latest source code from GitHub:

$ git clone https://github.com/zalando/system.git ./

3. Install Node.js dependencies:

$ npm install

And you're good to go!

Does it run?

After installing you should do a test run. If you used NPM, execute the systemapp command:

$ systemapp

If you installed using the script or manually, execute the index.js file under the installed folder:

$ node index.js

By default it runs on port 3003, so pop your browser to http://localhost:3003 and enjoy :-)
