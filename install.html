Installing the System App

Simply download the script, make it executable and run!

$ wget https://raw.github.com/zalando/system/master/install.sh
$ chmod +x install.sh
$ ./install.sh

Manual installation

If you want to install it manually, make sure you have these on your system:

  • Node.js (0.8.x +)
  • MongoDB (1.8.2 +)
  • ImageMagick (optional to generate map thumbnails)

Download the latest source code from GitHub:

$ git clone https://github.com/zalando/system.git ./

Install Node.js dependencies:

$ npm install

And you're good to go!
