  • ¶

    When user clicks a menu button, make it active and open the relevant tab.

        openTab: (menu) =>
        @$allMenus = $("a.menu-item") if not @$allMenus
        @$allTabs = $("div.admin-tab") if not @$allTabs

        @$allMenus.removeClass "active"
        $("#menu-#{menu}").addClass "active"
        @$allTabs.hide()
        $("#tab-#{menu}").show()
