    METHODS

  • ¶

    Run all rules. This will return the list of matched Alert Rules, or an empty array if no rule was matched (so no action to do).

        run: (contextValues) =>
        matchedRules = []

        if @active()
            _.each @rules().models, (rule) =>
                if rule.test(contextValues)
                    matchedRules.push(rule)

        if matchedRules.length > 0
            @trigger "#{@id}:enable", this
        else
            @trigger "#{@id}:disable", this

        return matchedRules
