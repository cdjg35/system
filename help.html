Need help?

If you're having problems with the System App...

On the server side

  1. First, make sure all the dependencies are properly installed (Node.js, MongoDB, etc).
  2. If you're using Windows... eerrrm, switch to Linux or Mac OS X.
  3. Can't access the app on your browser? Double check if the settings.web.port is correct.
  4. Can't save any data? Double check if the settings.database.connString is correct.

On the client side

  1. We recommend the latest version of Google Chrome - but other browser * might * work.
  2. If things are scrambled up or features stop working, reloading the page (usually pressing F5) helps a lot.
