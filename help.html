Need help?

If you're having problems with the System App...

On the server side

  1. Make sure all the dependencies (Node.js, MongoDB, etc) are properly installed.
  2. If you're using Windows... eerrrm, switch to Linux or Mac OS X :-)
  3. Can't access the app on your browser? Double check if the settings.web.port is correct.
  4. Can't save any data? Double check if the settings.database.connString is correct.

On the client side

  1. We recommend using the latest version of Google Chrome, even if other browser might work.
  2. If things are scrambled up or features stop working, reloading the page (usually pressing F5) might help.

Source code documentation

Sometimes it's good to understand what's going on behind the scenes so if you're interested in get details about the System App's source, please head on to the docs/ folder.
